quizzer

trasforma una lista di domande in un quiz interattivo

Istruzioni per l'uso:

  1. Procurati o prepara un documento con le domande su cui vuoi testarti.
  2. Chiedi ad un'AI di formattarlo come richiesto da questa app usando il prompt di seguito (copia il prompt e incollalo nella chat AI aggiungendo al prompt la lista delle domande in formato testo o allegando il file di origine).
  3. Scarica il file .yaml o copia il testo generato dall'AI e caricalo o incollalo in questa pagina.
  4. Scegli se vuoi vederle in ordine casuale o sequenziale.
  5. Inizia il tuo ripasso!
  6. Durante il quiz, puoi modificare le domande in tempo reale usando l'editor live (il quiz si aggiornerà automaticamente con le modifiche).
